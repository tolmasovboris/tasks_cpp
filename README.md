# Проекты на C++ (ООП)

## 📌 String

Собственная реализация строкового класса с:
- Динамическим выделением памяти
- Поддержкой базовых операций:
  - Конкатенация (`+`, `+=`)
  - Поиск подстрок
  - Ввод/вывод
  - Сравнение строк

## 🔢 BigInteger

Класс для работы с большими целыми числами:
- Хранение чисел произвольной длины
- Поддержка арифметических операций:
  - Сложение/вычитание (`+`, `-`)
  - Умножение/деление (`*`, `/`)
  - Остаток от деления (`%`)
- Операции сравнения
- Ввод/вывод в разных системах счисления

## 📐 Geometry

Библиотека для 2D геометрических вычислений:
- Работа с базовыми фигурами:
  - Точки
  - Линии/отрезки
  - Окружности
- Функционал:
  - Расчёт расстояний
  - Проверка пересечений
  - Вычисление площадей
  - Определение взаимного расположения

## 🧮 Matrix

Реализация матричных операций:
- Поддержка матриц разных размеров
- Основные операции:
  - Сложение/вычитание
  - Умножение (на число и матрицу)
  - Транспонирование
  - Вычисление определителя
- Обработка ошибок (несовместимые размеры и т.д.)

## Технологии
- C++17
- Объектно-ориентированное программирование
- Перегрузка операторов
- Шаблонное программирование (для Matrix)
